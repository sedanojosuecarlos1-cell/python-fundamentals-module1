# Proyecto Módulo 1 – Python Fundamentals

## Descripción

Aplicación interactiva desarrollada en **Streamlit** que integra los conceptos fundamentales de programación aprendidos en el Módulo 1 de la Especialización en Python for Analytics: variables, condicionales, estructuras de datos, funciones, programación funcional (`map` y `lambda`) y Programación Orientada a Objetos (POO).

---

## Estructura del rpoyecto

```
├── app.py              # Aplicación principal
├── requirements.txt    # Dependencias
└── README.md           # Documentación
```

## Instrucciones de ejecución

1. Instala las dependencias:
  ```bash
  pip install -r requirements.txt
  ```

2. Ejecuta la aplicación:
  ```bash
  python -m streamlit run app.py
  ```
